# Trello Clone
> Study clone project of Trello

## Demo
https://trelloclonereact.netlify.app

## Features
* Create lists
* Create cards
* Drag and drop cards between lists
* Add comments to cards (with timestamp)
* Add checklists to cards (with checklist items)
